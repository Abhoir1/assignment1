Parametric Functions ------------------------------------------------------------------------------- ![Torus Point Cloud](/home/ab/CMSC848F/assignment1/outputs/torus_video.gif) Implicit Surfaces ------------------------------------------------------------------------------- ![Torus Mesh](/home/ab/CMSC848F/assignment1/outputs/torusImplict_video.gif) Point Cloud vs mesh 1. **Rendering Speed**: Point clouds are usually faster to render because it renders individual points on the other hand meshes involves rendering traingles thats why relatively slow. 2. **Rendering Quality**: Point clouds usually have lower rendering quality because they do not have surface geometry. The object might not be smooth. On the other hand, meshes provide better rendering quality as they have surfaces and can be smooth. 3. **Ease of Use**: Point clouds just need position and color as parameters whereas meshes require vertices, faces, textures therefore can be more complex. 4. **Memory Usage**: Point clouds are memory efficient because they has limited number of points vs meshes can be more memory consuming.
